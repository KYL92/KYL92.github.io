# Publications Images

이 디렉토리는 Publications 페이지에서 사용되는 이미지들을 저장합니다.

## 필요한 이미지 파일들

### 메인 이미지
- `research-overview.png` - 연구 분야 개요 이미지

### 논문 관련 이미지
- `rppg-paper.png` - Remote PPG 논문 이미지
- `noise-assessment.png` - Noise Assessment 논문 이미지
- `cvpr-logo.png` - IEEE/CVF CVPR 로고

### 연구 분야 이미지
- `computer-vision.png` - Computer Vision 아이콘
- `affective-computing.png` - Affective Computing 아이콘
- `rppg.png` - Remote Photoplethysmography 아이콘
- `facial-expression.png` - Facial Expression Recognition 아이콘

## 이미지 권장 사양
- 크기: 80x80px (아이콘), 300x200px (논문 이미지)
- 형식: PNG 또는 SVG
- 스타일: 플랫 아이콘 또는 미니멀한 디자인

## 임시 사용 방법
실제 이미지가 없을 때는 다음과 같은 플레이스홀더를 사용할 수 있습니다:
- `https://via.placeholder.com/80x80/1976d2/ffffff?text=CV` (Computer Vision)
- `https://via.placeholder.com/80x80/7b1fa2/ffffff?text=AC` (Affective Computing)
- `https://via.placeholder.com/80x80/388e3c/ffffff?text=rPPG` (Remote PPG)
- `https://via.placeholder.com/80x80/f57c00/ffffff?text=FE` (Facial Expression)
